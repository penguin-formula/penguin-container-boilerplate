# Penguin API

API example to demonstrate docker, docker-compose and kubernetes

### Tech

Penguin API uses a number of open source projects to work properly:

* [node.js] 
* [Express] 
* [NestJS]
* [Mongodb]
* [Docker]
### Running

```sh
$ docker-compose up
```
